## 📝 核心指南：党政推文写作

### 一、推文写作核心原则
政治准确：严格遵循宣传口径，确保政策表述、术语使用符合官方规范
逻辑清晰：遵循「背景 — 内容 — 意义」的基本叙事结构
语言精炼：避免冗余表达，兼顾规范性、可读性与严肃性
格式统一：标题层级、标点与排版样式严格对标党政文稿要求

### 二、推文标题写作技巧
标题类型	示例	适用场景
政策解读型	「XX 政策」解读：XX 工作怎么干？	政策宣贯类推文
活动纪实型	XX 活动圆满举办：凝聚力量促发展	活动总结报道
经验分享型	XX 单位：以 XX 举措推动 XX 工作落地	典型经验推广

### 三、正文写作规范
开头：简要交代活动或政策背景，原则上控制在 100 字以内
主体：分点阐述核心内容，每段不超过 3 行，重点信息突出
结尾：总结意义或提出要求，形成首尾呼应，强化导向

### 四、排版与格式要求
字体：正文宋体 / 仿宋，标题黑体（标题二号，正文小四）
行距：正文行距 1.5 倍，段落间距约 2 字符
标点：使用全角标点，数字和英文采用半角
配图：主题贴合、来源清晰，推荐宽度 800px


## 📷 配套指南：活动摄影技巧

### 一、摄影核心要求
构图：突出主体，画面简洁，符合党政宣传审美
光线：优先自然光，室内活动避免面部阴影
角度：正式活动以正面或 45° 角为主
画质：分辨率不低于 1920×1080，画面清晰稳定

### 二、不同场景拍摄要点
会议场景
主席台全景与人物特写相结合
捕捉发言、讨论等关键瞬间
拍摄文件、笔记等细节内容
户外活动
结合环境与活动主题进行构图
以抓拍为主，避免明显摆拍
远景与近景结合展示规模与细节
素材整理
按「场景 — 人物 — 细节」分类归档
命名示例：XX 活动 - 会议现场 - 2026-01.jpg
保留高清原图，标注时间、地点与人物信息
